# UniConnect
## Connecting Students for Global Research Collaboration

UniConnect is a web platform that enables students and researchers from different universities to collaborate on research projects. The platform promotes knowledge sharing, innovation, and partnerships aligned with **SDG-17: Partnerships for the Goals**.

--------------------------------------------------

## Problem Statement

Research opportunities are often limited within individual universities. Students from other institutions cannot easily discover or contribute to ongoing research projects. This lack of collaboration limits innovation, knowledge sharing, and access to research opportunities.

--------------------------------------------------

## Solution

UniConnect provides a centralized platform where:

- Universities can publish research projects
- Students can discover and join projects
- Collaboration happens across universities
- Skills and knowledge can be shared globally

--------------------------------------------------

## Installation and Setup

1. Clone the repository

git clone https://github.com/your-username/uniconnect.git
cd uniconnect

2. Install dependencies

npm install

3. Run the server

node server.js

4. Open the application in your browser

http://localhost:3000/auth.html

--------------------------------------------------

## How It Works

1. User creates an account using the registration form
2. User data is stored in MongoDB
3. User logs in with email and password
4. After successful login, the user is redirected to the dashboard
